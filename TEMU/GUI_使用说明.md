# TEMU GUI 使用说明

## 概述

TEMU 图形化界面提供实时查看寄存器值和内存内容的可视化工具，支持通过按钮执行常用命令。

## 界面布局

### 顶部按钮栏
窗口顶部提供6个功能按钮：

- **Continue (c)** - 继续执行程序直到结束或遇到断点
- **Step (si 1)** - 单步执行1条指令
- **Step 10** - 执行10条指令
- **Info Reg** - 在终端显示所有寄存器信息
- **Memory** - 在终端显示内存内容（0x80000000开始）
- **Quit (q)** - 退出程序

### 寄存器显示区
- 显示32个通用寄存器（$0-$31）的值，4列布局
- 显示PC（程序计数器）寄存器的值
- 所有值以十六进制格式显示（0xXXXXXXXX）

### 内存显示区
- 显示从地址 0x80000000 开始的内存内容
- 每行显示一个地址及其对应的4个字（16字节）
- 自动适应窗口大小，显示尽可能多的内存行

## 使用方法

### 基本操作
1. **启动程序**：运行 `make run` 后，GUI窗口会自动打开
2. **点击按钮**：使用鼠标左键点击顶部按钮执行对应命令
3. **查看信息**：寄存器值和内存内容会实时更新显示
4. **调整窗口**：可以拖拽窗口边缘调整大小，内容会自动居中适应

### 实时更新
- 执行指令后，GUI会自动刷新显示最新的寄存器值和内存内容
- 点击按钮执行的命令会在终端以 `(temu) <command>` 格式显示

### 窗口操作
- **调整大小**：拖拽窗口边缘或点击最大化按钮
- **关闭窗口**：点击窗口关闭按钮（程序不会退出，仍可通过终端控制）

## 注意事项

1. **SDL2 依赖**：需要系统安装 SDL2 和 SDL2_ttf 库
2. **X11 服务器**：在 WSL2 环境下需要运行 X11 服务器（如 VcXsrv）
3. **终端同步**：GUI按钮操作会同步显示在终端，便于跟踪执行过程
4. **实时性**：GUI更新与指令执行同步，可实时观察程序状态变化

## 快捷键

GUI界面本身不提供快捷键，所有操作通过鼠标点击按钮完成。终端命令仍然可用，与GUI操作并行工作。

## 故障排除

- **窗口不显示**：检查 DISPLAY 环境变量和 X11 服务器状态
- **字体显示异常**：系统缺少字体文件，但不影响基本功能
- **按钮无响应**：确保鼠标点击在按钮区域内
